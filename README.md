# Library Management System

#### Kivy android app with login page | You can use it and add your functions !

> Includes .kv file

- Login page
- Signup page
- Book List page
- Book favorite page
- Backgroud image
- Action bar
- Search bar
- Profile page
- Dark theme

#### 

# Install

- clone

```bash
git clone https://github.com/futuresea-dev/LMS-app
```

- requiremets

```bash
pip install -r requirements.txt
```

- run

```bash
python3 main.py
```
